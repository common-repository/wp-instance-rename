On-demand easily renames your Wordpress.

== Description ==

Wordpress Rename plugin allows you to easily rename the complete Wordpress installation.
This plugin allows you to rename Wordpress database, Wordpress directory, change every necessary configuration file, easily from one page.
No more, manually changing the database, directory, updating configuration files etc.
This plugin will do every step for you.
NOTE: only Wordpress admins can use this plugin, beware it can easily corrupt the Wordpress installation in the wrong hands.

Some of the features:
 - Backup Wordpress database/directory
 - Rename Wordpress database/directory
 - Update configuration file
 - Update Wordpress URL and site URL
 - Download backup database in zip format
 - etc

== Installation ==

1. Extract wordpress_rename/ folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Update config.ini file, and set mysql folder location.
3. Optionally instead of setting mysql folder location, user can manually backup database, and supply the path to the .sql file. In the config.ini file or provide the path to the file in the form input on the page.

== Frequently Asked Questions ==

1. MySQL dump necessary? Yes :) But the user can manually backup database, and set the path to the .sql in the form on the page or through the config.ini file.

== Usage ==
1. Use from the "Settings" menu in Wordpress
2. Provide login information
3. Enter new Wordpress directory name
4. Enter new Wordpress database name
5. Select additional settings
6. Click "Update Name" and wait.
7. The procedure is done.Voilà :) The user can now verify if everything has been done correctly.
8. Optionally, if wordpress directory has been changed, the user will see a redirect link to the new website location.
9. Optionally, the user can download the database backup file.
